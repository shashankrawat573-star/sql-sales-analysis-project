# sql-sales-analysis-project
SQL project using SSMS for sales data analysis# SQL Sales Analysis Project

## Project Objective
To analyze sales data using SQL queries in SSMS and generate meaningful business insights.

**Project Overview**
This project contains SQL queries commonly used in real-world sales analysis:
Monthly sales summary,
Top customers by revenue,
Product sales analysis,
Revenue by city,
Customer and order filters,
CTE-based transformations,
Ranking cities by revenue,
Aggregation and grouping,
Payment mode analysis,
Date-based filtering

## Tools Used
- SQL Server Management Studio (SSMS)
- SQL Server
- Microsoft Excel (if used)

## Database Name: SalesAnalysisDB
Tables Included:
- customers
- products
- orders
- payments

## Key SQL Concepts
- JOINS
- GROUP BY
- WHERE
- ORDER BY
- Aggregate functions
- Date functions (MONTH, YEAR)

## Business Questions Solved
- Total sales by month
- February 2024 orders
- Top selling products
- Customer-wise purchases
